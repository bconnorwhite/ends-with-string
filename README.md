## Installation

```sh
yarn add ends-with-string
```

```sh
npm install ends-with-string
```

```sh
pnpm add ends-with-string
```

## API

```ts
import endsWith from "ends-with-string";

function endsWith(input: string | Buffer, string: string): boolean;
```
